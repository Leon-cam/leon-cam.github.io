Support

iUSBDebugger is a macOS utility for inspecting USB, USB-C, USB Power Delivery, Apple mobile device connectivity, and Thunderbolt / USB4 topology.

Supported Platform

macOS 14.0 or later.

Common Issues

  • If a device does not appear, disconnect and reconnect it, then refresh the app.
  • If a hub or adapter is in the path, test with a direct connection to isolate the issue.
  • Thunderbolt and USB4 topology may appear incomplete until negotiation fully settles.
  • Apple mobile device connector type is intentionally conservative when the operating system does not provide reliable proof.
